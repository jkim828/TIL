##### 1. Python 기본문법 by 애란쌤

- literal : 어떤 값을 표현하기 위한 표현법 (ex, 12, 1-2, tweleve)
- 리터럴 = 표현식, 변수 = 표현식, 표현식 + 표현식 = 표현식

  Animal_name = {

  }
